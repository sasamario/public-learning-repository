# Laravel コマンド一覧

■ コマンド一覧

```bash
php artisan list
```

■ ルート一覧

```bash
php artisan route:list
```

■ コントローラ作成

```bash
php artisan make:controller SampleController
```

※Docker コンテナ内で実行する際は、あらかじめ app フォルダの所有者を変更しておくこと

■ モデル作成

```bash
php artisan make:model Sample
```

※Docker コンテナ内で実行する際は、あらかじめ app フォルダの所有者を変更しておくこと

■tinker

```bash
# 起動
php artisan tinker

# 終了
exit
```

■ 設定キャッシュクリア
`.emv`や`config/*.php`を変更したのに反映されない場合

```bash
php artisan config:clear
```

■ キャッシュクリア
Cache のデータが古い場合

```bash
php artisan cache:clear
```

■ ルートキャッシュクリア
ルーティング変更が聞かない場合

```bash
php artisan route:clear
```
