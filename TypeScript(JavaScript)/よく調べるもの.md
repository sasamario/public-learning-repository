## ??（Null 合体演算子）

Null 合体演算子は、論理演算子の一種で、この演算子の左辺が「null」または「undefined」の場合に、右の値を返しそれ以外の場合は左の値を返す。

```js
// hoge = null,undefinedの場合、test = 'default'
// hoge = null,undefined以外の場合、test = hoge
const test = hoge ?? "default";
```
